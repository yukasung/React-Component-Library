# React Component Library

Docs for the internal React Component Library — reusable React + TypeScript
components styled with Tailwind CSS.

## Components

- [**InputNumber**](/components/input-number) — input สำหรับกรอกตัวเลข
  รองรับการจำกัดค่าต่ำสุด/สูงสุด (min/max) ปุ่มเพิ่ม-ลดค่าและปุ่มลูกศร (step)
  การจัดรูปแบบตัวเลข (format) และรองรับการใช้งานผ่านคีย์บอร์ดได้อย่างครบถ้วน
- [**InputDate**](/components/input-date) — input สำหรับกรอกวันที่
  รองรับทั้งการพิมพ์ข้อความและเลือกจากปฏิทินแบบ dropdown จำกัดช่วงวันที่
  ต่ำสุด/สูงสุด (min/max) และการจัดรูปแบบวันที่ (format)
