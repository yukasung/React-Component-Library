# InputNumber

InputNumber คือ input ที่ให้ผู้ใช้กรอกและแก้ไขได้เฉพาะตัวเลขเท่านั้น ค่าที่กรอกสามารถจัดรูปแบบหรือจำกัดให้อยู่ในช่วงที่กำหนดได้

<DefaultDemo />

## Import

```tsx
import { InputNumber } from '@your-scope/components'
```

<PropertySignature name="value" type="number | null" />

<InputNumber value={value} onChange={setValue} />

<PropertySignature name="placeholder" type="string" />

</Callout>

### isRequired

<PropertySignature name="isRequired" type="boolean" />

กำหนดค่าที่ระบุว่าค่าของ control ต้องเป็นตัวเลขเสมอ หรือสามารถตั้งเป็น null ได้ (โดยการลบเนื้อหาในช่องออกทั้งหมด)

ค่าเริ่มต้นของ property นี้คือ true

<IsRequiredDemo />

```tsx
<InputNumber isRequired={false} defaultValue={null} />
```

### min

<PropertySignature name="min" type="number" />

<Callout type="info">
  **Note**

  ไม่ระบุหมายถึงไม่จำกัดค่าต่ำสุด

### max

<PropertySignature name="max" type="number" />

กำหนดค่าสูงสุดที่ยอมรับได้ การพิมพ์ที่จะทำให้ค่าเกินนี้ถูกบล็อกทันที

<Callout type="info">
  **Note**

  ไม่ระบุหมายถึงไม่จำกัดค่าสูงสุด
</Callout>

<MinMaxDemo />

```tsx
<InputNumber min={0} max={100} defaultValue={50} />
```

### step

<PropertySignature name="step" type="number | null" />

กำหนดจำนวนที่จะเพิ่มหรือลดค่า value เมื่อผู้ใช้กดปุ่ม spinner

ค่าเริ่มต้นของ property นี้คือ null ซึ่งจะซ่อนปุ่ม spinner จาก control

<Callout type="info">
  **Note**

  step เป็นตัวกำหนดว่าจะแสดงปุ่ม spinner หรือไม่ ไม่มี prop แยกสำหรับเรื่องนี้

  ถ้าไม่ระบุ step ปุ่ม spinner, ปุ่มลูกศรบนคีย์บอร์ด และ handleWheel จะใช้งานไม่ได้ทั้งหมด เพราะไม่มีจำนวนให้ขยับค่า

  ถ้า step มีทศนิยมและไม่ได้ระบุ precision เอง ระบบจะคำนวณจำนวนทศนิยมที่แสดงให้อัตโนมัติ
</Callout>

<StepDemo />

```tsx
<InputNumber defaultValue={1} />
<InputNumber step={0.1} defaultValue={1} />
```

### format

<PropertySignature name="format" type="string" />

รูปแบบการแสดงผลตัวเลข เขียนด้วยตัวอักษร 1 ตัวบอกประเภท ตามด้วยจำนวนทศนิยมที่เป็นตัวเลือก (เช่น "n2", "C", "p0") เมื่อระบุจะแทนที่ precision เดิมทั้งหมด

รองรับครบทั้ง 9 ตัว:

| ตัวอักษร | ความหมาย | ตัวอย่าง (ค่า 1234.5) |
| :---: | --- | --- |
| "N" หรือ "n" | ตัวเลขมี comma คั่นหลักพัน | 1,234.50 |
| "C" หรือ "c" | สกุลเงิน (นำหน้าด้วย $) | $1,234.50 |
| "P" หรือ "p" | เปอร์เซ็นต์ (คูณค่าด้วย 100 แล้วต่อท้ายด้วย %) | 123,450.00% |
| "F" หรือ "f" | ทศนิยมธรรมดา ไม่มี comma | 1234.50 |
| "D" หรือ "d" | จำนวนเต็ม (ตัดทศนิยมทิ้ง) เติม 0 ด้านหน้าได้ | 1234 |
| "E" หรือ "e" | สัญกรณ์วิทยาศาสตร์ | 1.234500E+003 |
| "G" หรือ "g" | เลือกรูปแบบกะทัดรัดที่สุดระหว่างทศนิยมกับวิทยาศาสตร์ให้เอง | 1234.5 |
| "X" หรือ "x" | เลขฐาน 16 (ตัดทศนิยมทิ้งก่อน) | 4D2 |
| "R" หรือ "r" | ค่าที่แปลงกลับเป็นตัวเลขเดิมได้เป๊ะที่สุด | 1234.5 |

ตัวเลขที่ต่อจากตัวอักษร (เช่น "n2") กำหนดจำนวนทศนิยมที่แสดง ยกเว้น D และ X ที่หมายถึงจำนวนหลักขั้นต่ำแทน ไม่ระบุจะใช้ค่า default ของตัวอักษรนั้น ตัวพิมพ์เล็ก/ใหญ่ของตัวอักษร (เช่น "C" กับ "c") มีผลแค่ตัวพิมพ์ของผลลัพธ์ในบาง specifier (เช่น E/e, X/x) เท่านั้น ไม่มีผลกับความหมาย

<Callout type="info">
  **Note**

  ค่าติดลบแสดงด้วยเครื่องหมายลบนำหน้าเสมอ ยกเว้น "C" (สกุลเงิน) ที่ครอบด้วยวงเล็บแทน (เช่น -1234.5 แสดงเป็น "($1,234.50)") — การจัดรูปแบบเกิดขึ้นระหว่างพิมพ์จริง
</Callout>

<FormatDemo />

```tsx
<InputNumber format="n0" defaultValue={0} />
<InputNumber format="n2" defaultValue={0} />
<InputNumber format="c0" defaultValue={0} />
<InputNumber format="c2" defaultValue={0} />
<InputNumber format="p0" defaultValue={0} />
<InputNumber format="p2" defaultValue={0} />
```

### truncate

<PropertySignature name="truncate" type="boolean" />

กำหนดว่าจะตัดทศนิยมส่วนเกินทิ้งแทนการปัดเศษหรือไม่ เมื่อค่าที่พิมพ์มีทศนิยมมากกว่าที่ระบุใน precision หรือ format

ค่าเริ่มต้นของ property นี้คือ false (ปัดเศษตามปกติ)

<Callout type="info">
  **Note**

  เช่น precision เป็น 1 แล้วพิมพ์ 2.999 — ถ้า truncate เป็น false (ปกติ) จะปัดเป็น 3.0 แต่ถ้า truncate เป็น true จะตัดทิ้งเหลือ 2.9 แทน
</Callout>

<TruncateDemo />

```tsx
<InputNumber precision={1} step={1} defaultValue={0} />
<InputNumber precision={1} step={1} defaultValue={0} truncate />
```

### repeatButtons

<PropertySignature name="repeatButtons" type="boolean" />

กำหนดค่าที่ระบุว่าปุ่ม spinner ควรทำงานเป็นปุ่มทำซ้ำหรือไม่ โดยจะยิงค่าซ้ำต่อเนื่องตราบใดที่ยังกดปุ่มค้างไว้

ค่าเริ่มต้นของ property นี้คือ true

<Callout type="info">
  **Note**

  กดครั้งเดียวโดยไม่ค้างยังคงขยับ 1 step ตามปกติเสมอ ไม่ว่าตั้งค่านี้เป็นอะไร
</Callout>

<RepeatButtonsDemo />

```tsx
<InputNumber step={1} defaultValue={0} />
<InputNumber step={1} defaultValue={0} repeatButtons={false} />
```

### handleWheel

<PropertySignature name="handleWheel" type="boolean" />

กำหนดว่าจะให้เลื่อนล้อเมาส์ตอน focus อยู่ที่ฟิลด์เพื่อเพิ่ม/ลดค่าได้หรือไม่ ค่าเริ่มต้น (default): false

<Callout type="info">
  **Note**

  ทำงานเฉพาะตอนฟิลด์ focus อยู่เท่านั้น ไม่ทำงานถ้า isDisabled หรือ isReadOnly
</Callout>

<HandleWheelDemo />

```tsx
<InputNumber step={1} defaultValue={5} />
<InputNumber step={1} defaultValue={5} handleWheel />
```

### isReadOnly

<PropertySignature name="isReadOnly" type="boolean" />

เมื่อเป็น true ผู้ใช้แก้ไขค่าไม่ได้ แต่ยังคง focus เลือกข้อความ และคัดลอกค่าได้ตามปกติ

ค่าเริ่มต้น (default): false

<ReadOnlyDemo />

```tsx
<InputNumber step={1} defaultValue={10} />
<InputNumber step={1} defaultValue={10} isReadOnly />
```

### isDisabled

<PropertySignature name="isDisabled" type="boolean" />

เมื่อเป็น true ปิดการใช้งานทั้งหมด พิมพ์และ focus ไม่ได้ แสดงผลจางลงเพื่อสื่อว่าใช้งานไม่ได้

ค่าเริ่มต้น (default): false

<DisabledDemo />

```tsx
<InputNumber step={1} defaultValue={10} />
<InputNumber step={1} defaultValue={10} isDisabled />
```
